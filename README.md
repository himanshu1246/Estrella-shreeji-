# ESTRELLA by Shreeji Group

Welcome to the official repository for the **ESTRELLA by Shreeji Group** website. This is a real estate landing page showcasing premium 3, 4, and 5 BHK apartments located in the heart of Kharghar City, Navi Mumbai.

## 🌟 Project Highlights
* **Location:** Sector 33, Kharghar City, Navi Mumbai.
* **Property Types:** Super-sized 3, 4, and 5 BHK Apartments.
* **Pricing:** Starting from ₹ 3.25 Cr*.
* **Architecture:** Ground + 4 Level Podium Parking + 35 Storeys of sheer luxury.

## 🚀 Features & Implementation
* **Responsive Design:** Fully responsive UI to ensure seamless viewing across mobile, tablet, and desktop devices.
* **Interactive Elements:** Features custom forms, locked gallery previews, and direct PDF brochure downloads.
* **Asset Optimization:** Excludes raw high-res video files (`asset/video/*.mp4`) to keep the repository lightweight and compatible with standard Git LFS limits.
* **Styling:** Styled using modern CSS, Glider.js for smooth carousels, and clean typography.

## 💻 Local Setup
1. Clone this repository locally.
2. Open the directory using an IDE like VS Code.
3. Launch a local development server (e.g., using the "Live Server" extension in VS Code) to view the static HTML files properly without CORS issues.

## 📄 Pages Included
* `index.html`: The main landing and marketing page.
* `privacy-policy.html`: The privacy policy and terms of conditions.
* `thank-you.html`: A dedicated redirect page after successful lead-generation form submissions.
